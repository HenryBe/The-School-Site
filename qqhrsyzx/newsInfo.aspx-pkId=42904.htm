赢在起点

——如何做好新高一班主任

发言人：马忠哲

班主任的工作是辛苦的，也是神圣而有价值的。我个人认为做老师但没做过班主任永远都是教育生命的缺憾。首先对新从事班主任工作的老师表示祝贺，对继续任职班主任的教师道声辛苦和尊敬。

对于每个学生而言，从初中进入高中是人生的一个转折点。高中阶段是他们长身体、长知识的黄金时期，也是人生观、世界观、价值观形成的重要时期。而高一则是学生生活习惯、学习习惯养成的关键时期，俗话说：好习惯终身受益。

如何做好高一班主任工作谈点自己的感受。

一、完善自我，做自信的班主任。

只有自信的班主任，才能正确认识自己，了解自己的特长、优势和劣势，从而扬长避短，从宏观上把握班级发展态势，制订合适的班级计划。 自信的班主任，会永葆积极乐观的心态，从不利中发现有利。班主任工作的确存在艺术和方法，但是做一个好班主任并不难。只要自己认真、用心、付出你对学生真正的关爱，坐住板凳，耐住寂寞，那么你就会成为一个合格甚至优秀的班主任，得到学生认可的班主任，也会成为一个自信的班主任。（例如李晓丹）

二、踏实肯干，做实干型班主任。

1、迅速了解学生，看透学生档案，做好信息采集。每年开学我都做个信息采集表。信息采集表包括学生姓名，性别，出生年月、毕业学校、中考分数、兴趣爱好、特长、家庭基本信息、自己理想追求、是否想当班干部、想当什么班干部，这也作为了解学生的一个媒介。

（一）“勤看”：新生还未入学时，先要认真“查看”学生的档案，对学生的家庭住址、成员，有无特长等基本情况作一个大概了解。入学后，还要善于“察看”新生的言行，了解其个性。

（二）“勤问”：新生入学后，要更进一步深入了解，就需要“勤问”。问一些家庭和个人生活、学习方面的事如：毕业学校、家庭成员及其经济收入来源、是否适应学校生活，平时的爱好等等。其次，还可以通过询问来自同一所初中毕业学校的同学侧面了解其过去。

（三）“勤记”：作为高一新生班主任要及时了解每一位学生工作量很大，同时也容易忘记和混淆，因此班主任要把看到的、问到的新生情况及时整理，记录在案，最好在学期之初就给每一位学生建一个学生综合档案。此外，学生情况最好记在心中，做到心中有数。特别是新生入学时要在最短的时间内记住每一位学生的姓名，并要对上号。这样学生会认为你很重视他，从而为班级工作开展带来了便利。

（四）“勤督查”：由于新生相当一部分是独生子女，平时在家都是衣来伸手，饭来张口，生活、学习行为习惯差、生活自理能力差。因此，入学的第一个月就是重点抓学习、生活行为习惯的培养。

2、精心准备一个精彩、实在、凝练的开场白。你的第一天与学生见面很重要，学生会向很多人描述他第一天所见到的班主任。你的穿着、你的幽默、你的风格都会给学生留下深刻的印象。

4、让学生做好自我介绍，班主任作好记录，这是了解学生的第一步。（可以拿着信息采集表）为班级选好班干部做好准备。

5、制定学生姓名卡片，这有利于班主任、同学之间、科任与学生之间迅速认识。这个方法切实有效可行。

7、习惯的培养尤为关键，养成教育是高一的重中之重。例如早自习来了就进入自习状态。平时来了就保证安静等。这得需要班主任要来得早，9年班主任生涯，我不是夸自己，我每天基本上是比较早到校的。举例例如迟到你罚他在外边站上一堂课不会解决他迟到的问题，第一次可以一笑而过，第二次可以简单教育，第三次严厉批评杀一儆百。在此过程力求让他提高认识。

8、去掉学生的“娇骄二气”

   如何去掉学生的娇骄二气？

   要有大局观念、集体意识，不能再成为公子哥小公主

   要养成朴素的作风、朴实的风格。校服发下来就穿校服。衣服干净就好。

   要养成不吃小食品，不喝可乐等习惯多发现正面的例子去引导。

   要做好家长的思想工作，控制学生过多零用钱

   要学会从小事做起，重新定位，敢于面对挫折，通过努力拼搏学习

   要学会体谅他人，懂得感恩等

12、认识高中课程安排的特点以及高中学生学习的特点。

三、关爱学生 做平民型班主任

常言道：“亲其师，信其道”。平易近人，对学生和蔼可亲，充满爱心的班主任，总是深受学生的欢迎，也最容易走近学生的心灵。

作为高一新生，来自不同的学校，彼此互不熟悉，加上环境陌生，很容易产生孤独感、不适应感。这时，班主任嘘寒问暖的关爱，或是帮助解决生活中的一些问题，会让学生很受感动。再说新同学来到一个陌生的学校，特别希望能得到班主任老师的“厚爱”，我们应抓住学生的这一心理，及时主动地找学生交流。交流的形式，可多种多样，可通过随笔的形式，，班主任可在随笔上多给他们写肯定、鼓励性的话语。也可通过谈心的方式。借此机会了解一下学生进校后对学校和老师的要求，生活上有什么问题需要解决，给予学生最大程度的帮助。谈话时，一定要注意语气要温和，随意些，不要过于严肃，场合可在教室，操场，时间可在课间等。总之，不要让学生有拘束感，紧张感。由于我很注意这些，我很少在办公室和走廊找学生谈心。可见，只要建立了良好的师生关系，主动亲近学生，取得了学生的信任，就能及时化解学生的心结，做学生思想工作也会事半功倍。

四、发挥班干部核心作用 做管理型班主任

一个优秀的班集体，不仅有一个优秀的班主任，还会有一个强有力的班干部队伍。作为高中生，有一定的组织和自我管理能力，如何充分地挖掘和培养学生的潜能，既是建设一个班级的需要，也是培养学生素质教育的一个方面。记得马卡连柯曾经说过：“教育了集体，团结了集体之后，集体自身就能成为一种很大的教育力量”。在这个集体之中，班干部起着核心作用。刚刚开学，老师对学生的情况了解不多，这就要求我们特别是班主任要学会观察，善于发现人才。学期初可以试行班干部流动制，让多数同学都试一试，通过师生的共同考察过近一个月的相互接触和了解，大家彼此之间不再陌生，我们采取竞选的方法，把那些工作能力强，责任感强，愿意为大家服务的同学推上干部的位置。这样做的好处有三： ( 一 ) 避免了仅通过民主投票产生班干部，把那些人缘关系好但不愿意为大家服务的人推上了干部的位子，导致班级建设受到影响。 ( 二 ) 这些干部都是同学们自己再三比较、权衡后推举出来的，因此干部同学工作过程中遇到的阻力较少。 ( 三 ) 通过竞选，可以激发同学们的进取心，可以让竞选失败的同学经受挫折，让他们发现不足，完善自我。此外，选定好班干部后，不能立即放手，还必须加强对班干部的培训，指导他们如何开展工作，如何树立班干部的威信等。

五、加强学习 、研究 ，做开拓型班主任

有人说过，一流的班主任是用思想去引领学生；二流的班主任是靠制度去管理学生。这话是有一定道理的。平时，我们经常发现部分学生对老师的教育不买帐，有反感甚至是对立的情绪，从本质上讲，大多数是由于我们教育方式、方法不当或思想观念的冲突。因此我们多花时间来研究学生是很有必要的，可这些方法、新的观念从何而来呢？

我以为（一）我们平时要善于总结，善于思考，找出其中的真正原因。再针对他们所处的年龄阶段和心理特点，多思考，多想些办法，找到适合他们的教育方法。（二）我们的班主任之间要加强彼此交流和合作，相互取长补短。（三）此外，我们自身要加强教育理论业务学习，从中获得新的观念，新的启迪。如我曾经对班级德育评比扣分居高不下的现象进行过研究，经过调查，我发现原因是一部分学生无所谓、缺乏团队精神。如何采取措施加强班级的团队精神，提高班级的凝聚力和向心力呢？我以为除了充分利用一些德育素材，加强团队精神教育。另一个有效的办法就是开展丰富多彩的班级活动。班主任可以指导班干部开展诸如主题班会、班级篮球赛校运动会及各项文艺活动。通过这些活动，既可以促进学生之间的相互了解，增进友谊，形成了一种团结向上，积极进取的氛围，把人心紧紧地团结在一起。一旦有了这种精神，班级就有了前进的希望和动力，就会永往直前，战无不胜。

　　俗话说：“好的开端是成功的一半”。对于班级而言也是一样的。高一是高中阶段的“花样年华”。如果不能抓住最初的绝好时机进行引导、教育和整合，不能在短时间内树立起班级正气，提高班级凝聚力和向心力，那么会给班级后继工作的开展带来很多麻烦和困难。因此，作为班级“灵魂”人物的班主任就需要多动脑，勤思考，根据班级的实际情况，及时调整工作思路、方法，和工作重心。唯有如此，才能保证班级处于不断的进步之中！
